# Valentines-Letter

Find My Heart

A small, cozy Valentine mini game and animated love letter made just for fun.

Click through the gifts and surprises to find all three hidden hearts, unlock the envelope, and open a sweet little message at the end. The experience is designed to feel playful, soft, and magical from start to finish.

Live site
[https://anishanawar.github.io/Valentines-Letter/](https://anishanawar.github.io/Valentines-Letter/)

## What this is

This project is a single page interactive experience built with simple front end tools.

It combines a short heart finding game with an animated envelope and a handwritten style letter reveal.

## Features

* Hidden heart mini game
* Animated envelope opening sequence
* Sparkle and confetti effects
* Soft floating background shapes
* Letter view with custom fonts and gentle transitions


## How to run locally

Just open:

```text
index.html
```

in any modern browser.

No setup, no build steps, and no dependencies.
